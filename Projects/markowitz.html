Markowitz Portfolio Optimization

Implemented quantitative portfolio optimization with Markowitz mean-variance model :

  • Application of the Markowitz mean-variance model for optimal asset allocation
  • Simulation of thousands of random portfolios (Monte Carlo) to estimate the efficient frontier
  • Computation of the covariance matrix and expected returns using historical market data retrieved with yfinance
  • Identification of the minimum variance portfolio and the tangency (max Sharpe) portfolio, along with visualization of the efficient frontier
